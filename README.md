# Rock-Paper-Scissors-Lizard-Spock

Rock, Paper, Scissors, Lizard, Spock

# Goals

 - Design a website featuring a game of 'Rock, Paper, Scissors, Lizard, Spock'
 - The website should be interactive through its use of Javascript
